The technologies employed in my application :

- Kotlin
- Jetpack Compose
- Material Design 3
- MVVM with Clean Architecture
- Dependency Injection with Hilt
- Navigation Component
- Android Jetpack
- Coroutines & Flow
- Firebase -> Authentication , Firestore
- Retrofit
- Room Database
- Coil
